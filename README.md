# -Door-Locker-Security-System-
a door locker system using 2 ATmega32 one as HIMI_ECU and the other one as Control_ECU

## Main Components
- ATmega-32 
- 2*16 LCD
- Keypad
- Buzzer
- DC motor
- EEPROM

## Drivers of HIMI_ECU
- Keypad
- LCD
- GPIO
- Timer1
- UART

## Drivers of CONTROL_ECU
- I2C
- External EEPROM
- GPIO
- Timer1
- UART
- Buzzer
- DC motor
